# 宝塔面板清理存储

宝塔面板清理存储优先用**面板可视化操作**，安全高效；空间紧张时用**SSH 命令**深度清理，再配合**自动清理**避免反复爆满。

### 一、面板可视化清理（新手首选）

#### 1. 清理网站 / 服务日志

- 网站日志：**网站 → 选择站点 → 清理日志**（一键清 Nginx/Apache 访问 / 错误日志）
- 面板 / 系统日志：**软件商店 → 安装「日志清理工具」→ 运行清理**
- 面板日志目录：`/www/server/panel/logs`（可手动删除旧日志）

#### 2. 清理备份文件（最占空间）

- 路径：**文件 → /www/backup**（网站 / 数据库备份集中存放）
- 操作：删除过期备份 → 进入**回收站 → 清空回收站**（不删回收站不释放空间）
- 建议：保留最近 3–7 份备份，其余删除

#### 3. 清理系统缓存 / 临时文件

- 面板：**软件商店 → 系统工具 → 清理系统缓存 / 清理 YUM/APT 缓存**
- 网站缓存：**网站 → 站点设置 → 清理缓存**（如 WordPress、静态缓存）

#### 4. 面板一键清理（快速）

- SSH / 面板终端执行：

```bash
cd /www/server/panel && python tools.pyc clear
```

- 作用：一键清**网站日志、PHP 会话、系统邮件、临时文件**

### 二、SSH 命令深度清理（空间爆满时）

#### 1. 清空回收站（必做）

```bash
rm -rf /www/Recycle_bin/*
```

#### 2. 清理 Nginx/Apache 日志

```bash
# 清空所有日志（安全）
truncate -s 0 /www/wwwlogs/*.log
# 或删除旧日志
rm -f /www/wwwlogs/*.log
```

#### 3. 清理 MySQL 二进制日志（谨慎）

```bash
# 清理3天前binlog（推荐）
mysql -e "PURGE BINARY LOGS BEFORE DATE(NOW() - INTERVAL 3 DAY);"
# 极端情况（停止数据库后清理，不推荐）
/etc/init.d/mysqld stop
rm -f /www/server/data/ib_logfile* /www/server/data/mysql-bin.*
/etc/init.d/mysqld start
```

#### 4. 清理 PHP 会话 / 临时文件

```bash
# 清理7天前PHP会话
find /tmp -name "sess_*" -mtime +7 -delete
# 清理Nginx临时缓存
rm -rf /www/server/nginx/proxy_temp_dir/*
```

#### 5. 查找并删除大文件

```bash
# 查找>100MB文件并排序
find / -type f -size +100M -exec du -h {} \; | sort -rh | head -20
```

#### 6. 查看磁盘占用

```bash
df -h  # 看整体占用
du -sh /www/*  # 看/www目录各文件夹大小
```

### 三、自动清理（预防爆满）

#### 1. 日志轮转（logrotate）

- 编辑：`vi /etc/logrotate.d/bt`
- 示例配置：

```plaintext
/www/wwwlogs/*.log {
    daily
    rotate 7
    compress
    missingok
    notifempty
    create 0640 www www
}
```

#### 2. 定时清理任务（面板计划任务）

- 每周一 3 点清理 7 天前日志 / 备份：

```bash
# 清理日志
find /www/wwwlogs -name "*.log" -mtime +7 -delete
# 清理备份
find /www/backup -type f -mtime +7 -delete
```

### 四、常见占用大户与安全提醒

|        目录        |      占用原因       |          清理建议           |
| :----------------: | :-----------------: | :-------------------------: |
|   `/www/wwwlogs`   | 网站访问 / 错误日志 |     每周清理，保留 7 天     |
|   `/www/backup`    |  网站 / 数据库备份  |   保留最近 3 份，其余删除   |
| `/www/server/data` | MySQL 数据 /binlog  | 定期清 binlog，不删 ibdata1 |
| `/www/Recycle_bin` |     面板回收站      |          定期清空           |
|       `/tmp`       |    系统临时文件     |     自动清理，保留 1 天     |

**安全提醒**：

- 删除前**务必备份**重要数据
- 不删`/www/server`下核心程序文件
- MySQL binlog 清理前确认无需主从同步
